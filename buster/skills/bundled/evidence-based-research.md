For an evidence-focused question:

1. Plan sub-questions before searching.
2. Collect multiple independent sources per sub-question.
3. Classify each finding: supported (multiple sources agree), single-source,
   conflicting, inference, or unknown.
4. Record conflicts explicitly.
5. Only label a finding "supported" when at least two independent sources agree.
6. Produce a report with Summary, Findings (labelled), Conflicts,
   Recommendations, and Sources.

Never claim a source is verified when it has only been retrieved.
