Given an existing research project:

1. Review the saved sources and notes.
2. Organize findings by theme.
3. Separate supported findings, single-source claims, conflicts, and unknowns.
4. Convert conclusions into concrete recommendations.
5. Produce a durable Markdown report stored locally.
